# Build-a-Book-Catalog-Table
A basic HTML project that showcases how to build an accessible book catalog using semantic table elements. It lists book details in a structured format and includes a dynamic footer summary.
Book Catalog Table
Description

This project is a simple Book Catalog web page built using semantic HTML. It demonstrates how to properly structure and display tabular data using HTML table elements such as table, thead, tbody, and tfoot.

The catalog lists book information including title, author, genre, and publication year. A footer row summarizes the total number of books displayed in the table.

This project focuses on clean markup, accessibility, and best practices for presenting structured data without using CSS or JavaScript.

Features

Semantic HTML table structure

Table header for column labels

Table body with multiple book entries

Table footer displaying total book count

Accessible and readable markup
